# bike_derby

Motor: Nema17 Stepper Motor
Motor Driver: DRV8825

4x4 Matrix Keyboard

+------+------+------+------+
|  1   |  2   |  3   |  A   |   Row 1 → Pin 38
+------+------+------+------+
|  4   |  5   |  6   |  B   |   Row 2 → Pin 40
+------+------+------+------+
|  7   |  8   |  9   |  C   |   Row 3 → Pin 42
+------+------+------+------+
|  *   |  0   |  #   |  D   |   Row 4 → Pin 44
+------+------+------+------+
   ↑      ↑      ↑      ↑
 C1(46) C2(48) C3(50) C4(52)
